# Irfan's Neovim Config 🔥

My personal Neovim configuration built on top of kickstart.nvim

## Features
- 🎨 Gruvbox theme with transparent background
- 📁 Neo-tree file explorer with recycle bin support
- 📋 Bufferline with LSP diagnostics
- 🚀 LSP support for Python, JS/TS, C/C++, C#, Lua, HTML/CSS, SQL
- ✨ Auto completion with blink.nvim
- 🔭 Telescope fuzzy finder
- 🌳 Treesitter syntax highlighting
- 💻 Alpha dashboard with custom header
- ⚡ Noice.nvim command UI
- 🎯 Custom keybinds

## Keybinds
| Key | Action |
|-----|--------|
| `<leader>e` | Toggle Neo-tree |
| `<leader>ff` | Find files |
| `<leader>ci` | Open init.lua |
| `<leader>cc` | Search config files |
| `<leader>cd` | CD to current file |
| `<S-h>` / `<S-l>` | Prev/Next buffer |
| `zz` | Close buffer |
| `<F5>` | Run current file |
| `<C-s>` | Save file |

## Installation
```bash
git clone https://github.com/irfanAkhtar1/neovimConfig.git ~/.config/nvim
nvim
```

## Requirements
- Neovim 0.10+
